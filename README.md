## 🛠️ Core Skills & Expertise

**Hardware & Installation**
Rack & stack, server installation & decommissioning (Dell, HP, Supermicro), component-level replacement & break/fix — SSD, HDD, NIC cards, DDR4/DDR5 RAM

**Out-of-Band Management**
iDRAC (Dell), iLO (HP), IPMI — remote power cycling, SOL console access, hardware health monitoring, firmware & BIOS configuration

**Cabling & Connectivity**
Fiber optic and copper patching, cross-connect installation, structured cabling, end-to-end labelling, OTDR testing & cleaning

**Monitoring & Operations**
Device health monitoring & troubleshooting (server, network, power), DCIM monitoring, capacity tracking, infrastructure documentation

**Ticketing & Asset Management**
Incident and change management via ServiceNow, asset documentation & inventory management

**Networking Basics**
TCP/IP, VLANs, basic network troubleshooting — comfortable in Linux and Windows environments
